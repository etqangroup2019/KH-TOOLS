## English

**Version:** 1.99 
**Blender Compatibility:** 4.0.0+ 

### Overview
KH-Tools is a comprehensive productivity suite for Blender designed to streamline the workflow for architectural visualization, environment design, and general 3D creation. It integrates multiple specialized managers and tools into a single, cohesive interface.

### New Features
- **Blender 5.0 Support**: Fully optimized and tested for Blender 5.0, including UI enhancements and API compatibility.
- **Improved Performance**: Faster loading times and more efficient memory management.

### Main Features

#### 1. Scatter System (KH-Scatter)
- Efficiently scatter objects on surfaces using Geometry Nodes or Particle Systems.
- Default display set to **BOUNDS** for improved viewport performance.
- Easy-to-use toggle for switching between performance and full detail display.
- Specialized presets for Grass, Leaves, Ivy, and Rocks.

#### 2. Camera Manager
- Manage multiple cameras easily within the scene.
- Quick access to camera settings and focal lengths.

#### 3. Light Manager & Light Mix
- Centralized control for all lights in the scene.
- **Light Mix** support for adjusting intensities and colors of light groups post-render (similar to Corona/V-Ray).
- Support for HDRI/Dome light setup.

#### 4. Asset Maker & Manager
- Quickly mark and organize objects as assets.
- Streamlined workflow for importing and managing assets across projects.

#### 5. Memory & Render Settings
- Optimized render presets to balance quality and speed.
- Quick access to Cycles and EEVEE engine settings.

#### 6. SketchUp Import Tools
- Advanced tools for importing and optimizing SketchUp models for Blender.
- Drag and drop support for `.skp` files.

#### 7. Material Tools
- Material management and quick assignment tools.

### Download
- [Download Latest Release](https://github.com/etqangroup2019/KH-TOOLS/releases/latest)

### Installation
1. Download the repository as a ZIP.
2. In Blender, go to `Edit > Preferences > Add-ons > Install`.
3. Select the ZIP file.
4. Enable `KH-Tools` from the list.

## العربية (Arabic)

**الإصدار:** 1.99  
**متوافق مع بلندر:** 4.0.0+ 

### نظرة عامة
تعتبر أدوات KH حزمة إنتاجية شاملة لبلندر، مصممة لتبسيط سير العمل في التصور المعماري، وتصميم البيئات، والإنشاء ثلاثي الأبعاد بشكل عام. تدمج هذه الحزمة العديد من الأدوات والمديرين المتخصصين في واجهة واحدة متماسكة.

### المميزات الجديدة
- **دعم بلندر 5.0**: تم تحسينه واختباره بالكامل ليعمل مع بلندر 5.0، بما في ذلك تحسينات الواجهة وتوافق الـ API.
- **تحسين الأداء**: سرعة أكبر في التحميل وإدارة أكثر كفاءة لذاكرة البرنامج.

### المميزات الرئيسية

#### 1. نظام التوزيع (KH-Scatter)
- توزيع الكائنات بكفاءة على الأسطح باستخدام "Geometry Nodes" أو أنظمة الجسيمات "Particle Systems".
- العرض الافتراضي مضبوط على **BOUNDS** (الصناديق المحيطة) لتحسين أداء واجهة العرض.
- زر تبديل سهل الاستخدام للتحويل بين نمط الأداء ونمط التفاصيل الكاملة.
- إعدادات مسبقة متخصصة للعشب، الأوراق، اللبلاب، والصخور.

#### 2. مدير الكاميرات
- إدارة الكاميرات المتعددة بسهولة داخل المشهد.
- وصول سريع لإعدادات الكاميرا والبعد البؤري.

#### 3. مدير الإضاءة و Light Mix
- تحكم مركزي لجميع الأضواء في المشهد.
- دعم **Light Mix** لتعديل شدات وألوان مجموعات الإضاءة بعد الرندر (مشابه لـ Corona/V-Ray).
- دعم إعدادات إضاءة HDRI و Dome.

#### 4. صانع ومدير الأصول (Assets)
- تمييز وتنظيم الكائنات كأصول (Assets) بسرعة.
- سير عمل مبسط لاستيراد وإدارة الأصول عبر المشاريع المختلفة.

#### 5. إعدادات الذاكرة والرندر
- إعدادات رندر مسبقة ومحسنة لتحقيق التوازن بين الجودة والسرعة.
- وصول سريع لإعدادات محركي Cycles و EEVEE.

#### 6. أدوات استيراد SketchUp
- أدوات متقدمة لاستيراد وتحسين نماذج SketchUp لتناسب Blender.
- دعم السحب والإفلات لملفات `.skp`.

#### 7. أدوات المواد (Materials)
- أدوات إدارة المواد والتعيين السريع.
